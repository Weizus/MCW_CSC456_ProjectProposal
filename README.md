# Pitch 
It feels as if students need seem to need to build revolutionary proofs of concept, present some incredible research paper at just as incredible conferences like NeurIPS or ICML. All this to have a chance at a future in computer science. I feel like a lot of us, maybe its just me, are lacking some super projects that sound amazing on paper, especially if we can get that project accepted by some large entity like the NY Government or CUNY/SUNY. The revolutionary idea is: <b>ID Verification utilizing blockchain technology.</b> 

# Description
A system either centralized or decentralized that allows for a number of people's data to be securely stashed into a form of digital wallet that gives them or an entity control of what data is shown at any given moment; Specific sharing of data, by this I mean instead of sending an entire file on someone, we can easily display just a state id or degree to specific entities/people. 

A system like this can also make traversing the digital landscape easier for students, members of government, or corporate settings. For students, imagine your college giving you both a physical and official, credited, one of one, digital degree that you can access at any time and can be easily sent to any employer or university. For members of government, we can guarentee they have access to only what they're allowed to see; For any specific amount of time. Corporate benefits from this byproduct also and in addition employees can find use with a digital ID they can swipe and access from their phones using some NFC add-on to the ecosystem.
